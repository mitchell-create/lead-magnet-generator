## How to Find Exact Values

**Recommended:** Check Prospeo API documentation or test with your API key:

1. Try a search with `industry=Technology`
2. Check the response for industry values
3. Use the values that appear in results

**Common format:** Industry names are typically capitalized, single words or phrases like "Technology", "Retail", "Financial Services"

---

## Tips

- Industry values may be case-sensitive
- Some may require exact matching
- Try variations if a value doesn't work
- Common values: Technology, Retail, Healthcare, Financial Services, Manufacturing
